Enigma Suite - Calculations

Start screen

In the module Calculations, Enigma provides several small calculators.

The current version supports two of these calculators:

  • Julian Day Number: calculates the Julian Day for Ephemeris Time and for Universal Time, and also the value for Delta T.
  • Obliquity: the obliquity of the earth's axis, both the mean obliquity and the true obliquity.

The screen shows an overview of the available calculators, a short description, and a button to start the calculator.

You can use only one of the calculators at the same time: the current screen becomes inaccessible until you finish the option you are working on.

The Cancel button closes this screen.
