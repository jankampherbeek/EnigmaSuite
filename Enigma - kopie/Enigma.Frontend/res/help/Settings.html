Settings

Settings define the location of files that Enigma uses. In version 0.1 these locations are fixed.

The current locations are:

  • Location of data files: c:\enigma\data Data files contain data, currently only in csv-format, that can be used by projects.
  • Location of project files: c:\enigma\project Project files include files with description of projects and files that are converted to Json format.
  • Location of export files: c:\enigma\export Files that you export, currently only in csv-format.
  • Location of Swiss Ephemeris files: c:\enigma\se The files as supplied by the Swiss Ephemeris that contain the data for the calculations. These files will (almost) never change.

Csv (comma delimited values) is the format that is used by spreadsheet software like Microsoft Excel and Libre Office Calc to export data.

Json is a format that is mostly used by software but can be read by humans.
