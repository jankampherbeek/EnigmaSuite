Calculations: Mean and true obliquity of the earth's axis

You can calculate the Mean Obliquity and the True Obliquity for a given date. The true obliquity takes disturbances due to nutation into account and changes less smoothly than the mean obliquity.

Data input

You need to enter the date, the obliquity is calculated for 0:00 UT for this date.

The format for the date is yyyy/mm/dd. You can skip leading zero's, both 2022/05/22 and 2022/5/22 will be accepted.

Select either the Julian or the Gregorian calendar. Default the Gregorian calendar is selected as it is always applicable for current dates.

Select a yearcount. For dates in CE there is no difference with the astronomical yearcount. For years BCE there is a difference of 1 year. The historical year 0 does not exist but is indicated as -1 BCE. Astronomical, this is the year 0. The historical year -2 is the same as the astronomical year -1.

Calculation results

After entering the required data you can click the button Calculate. If your input did not contain errors, you will see the results of the calculation in the bottom part of the screen. If Enigma did find errors, the input field(s) with the error will return yellow.

Reset

Click the button Reset to clear all entered fields and the calculated results. You can also change the input and just press Calculate again.

Back to the Calculation overview

Click the button Close to close the screen for the calculation of the Obliquity and to return to the overview screen for calculations.
