Enigma - Charts

This is the main screen for working with charts. You can add new charts, retrieve charts from the database, show a chart wheel, analyze the charts and use progressive techniques. And you can also change the configuration that influences the calculation and analysis of the charts.

General information

Under the heading Charts, you see information about the charts in the database: the number of charts and the last added chart. And you see the chart that is currently selected, if any.

Available charts

Under the heading Available charts, you see an overview of calculated charts and charts retrieved from the database. Initially, this table is empty.

Menu

The menu gives you access to all functions. Some functionality is also available via the buttons on the right.

The menu option General - Settings shows the current location of files that Enigma uses.

Via General - Configuration you can show and change the configuration. If you change the configuration, Enigma removes the content in the table under Available Charts. Screens that contain chart-wheels, positions or analyzed data (like aspects) will remain visible.

General - Close Charts, or the button Close, terminates the session for Charts.

The menu option Charts - New Chart, or the button New, will open a screen where you can enter the details for a new chart.

Charts - Search for Chart, or the button Search, opens a screen where you can search in the database for existing charts.

Charts - Delete chart, or the button Delete, will remove the current chart from the database but not from the Available charts.

Charts - Show wheel, or the button Wheel, draws a graphic presentation for the current chart.

Charts - Show positions, or the button Positions, shows tables with all available astronomical positions for the current chart.

The menu option Analysis contains several possibilities to analyse a chart. You need to select a chart before you can access these possibilities.

Analysis - Aspects presents an overview of actually formed aspects for the current chart.

Analysis - Harmonics shows an interactive screen where you can calculate harmonics for the current chart.

Analysis - Midpoints shows all midpoints and separately all occupied midpoints.

The option Progressive shows primary directions, an OOB-calendar and other progressive techniques.

Progressive - Primary Directions opens a screen for the calculation of primary directions.

Progressive - OOB calendar shows a screen with the OOB positions for 120 years from birth.

Progressive - Transits, sec. dir. and symbolic dir. opens a screen where you can define events and calculate the corresponding progressive positions in transits, secundary directions, and symbolic directions.

Help - About Enigma Charts shows some information about this Enigma module.

Help - Help for this window, or the button Help, shows the text you are currently reading.

Help - User Manual opens a PDF with the user manual.

Please note that the availability of menu items and buttons depends on the availability of a selected chart, a selected event or date range, and on other windows being opened.
