New project

Before you can create a project, a datafile already needs to have been imported. If you did not yet import a datafile, please do so now. Menu: Data - Import Data.

Description of the project

To define the project you need to provide a name and a description. Both fields are obligatory.

  • In Name for project you enter the name you want to use for your project.
  • In Description you can enter additional information about the project, typically a short description of your intended research.

Control group

Defining a project includes the creation of data for a control group. Currently you can only select Standard shifting of location, date and time. In future releases, Enigma will support additional methods to create a controlgroup.

Multiplication for control group

Select one of the available values as multiplication factor for the controlgroup. You can select either 1 (no multiplication), 10, 100 or 1000. By defining a multiplication factor you can enlarge the controlgroup.

Datafile to be used

Select a datafile from the list of available datafiles. If the list is empty you need to define a datafile first. Menu: Data - Import Data.

Saving the project

If you click the button Save, Enigma will check your input and underline incorrect input fields with a red color. If there are no errors, a folder structure for this project, a datafile and a control file are generated. If during this process an error occurs, you will receive a popup about the error. Otherwise you will receive a confirmation popup. After closing this popup you will return to the main screen.
