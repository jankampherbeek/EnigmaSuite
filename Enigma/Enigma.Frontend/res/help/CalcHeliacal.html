Heliacal risings and settings

Heliacal risings and settings were an important technique in ancient astrology. Enigma supports the calculation of these events for the visible planets (including the Moon). Fixed stars are not yet supported.

Celestial object

Select Moon, Mercury, Venus, Mars, Jupiter or Saturn.

Type of event

Select one of the following event types:

  • Heliacal rising, the first visibility of a celestial object before sunrise.
  • Heliacal setting, the last visibility of a celestial object after sunset.
  • Evening first (Mercury, Venus, Moon), the first visibility after sunset. Only possible for Mercury, Venus and Moon.
  • Morning last (Mercury, Venus, Moon), the last visibility after sunrise. Only possible for Mercury, Venus and Moon.

Start date

Enigma calculates the heliacal events after the start date. Enter the start date in the format yyyy/mm/dd and select the correct calendar. The date is always assumed to be an astronomical date.

Longitude

Enter the geographic longitude in the format ddd:mm:ss and select East or West.

Latitude

Enter the geographic latitude in the format dd:mm:ss and select North or South.

Altitude in meters

For enhanced precision, Enigma asks you to enter the altitude. You need to enter the values in meters. Negative values are allowed but do not use fractions. The altitude should be larger than -420 meter and smaller than 8850 meter. In case you are wondering: these are respectively the lowest point on Earth (borders of the Dead Sea) and the highest point on earth (the peak of the Mount Everest with a small margin added for eye sight).

See the user manual to learn more about finding the correct altitude.

Calculate

Click the button Calculate to start the calculation. If you did not enter all required information, or if you used a wrong format, you will get a popup window with an explanation of what is wrong. You will need to fix the input to start the comparison.
