Parallels

This window contains one table that shows all parallels and contra-parallels that are within orb. Enigma shows only the points that you defined in the configuration.

The table comprises 7 columns:
  • Name of the first point.
  • Glyph of the first point (if available).
  • A glyph for either a parallel or a contra-parallel.
  • Name of the second point.
  • Glyph of the second point (if available).
  • The actual orb.
  • The exactness of the aspect as a percentage. The higher the percentage, the smaller the orb.
