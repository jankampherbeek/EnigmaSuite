Configuration progressive

This configuration contains your preferences for progressive techniques. Just like the general configuration, Enigma provides default values at install time, but you can override these values.

You can define the settings for three progressive techniques, each in their own tabbed page:

  • Transits in the tab with the same name.
  • Secondary directions in the tab Sec dir.
  • Symbolic directions in the tab Sym Dir.
  • Primary directions in the tab Prim Dir.

After changing the configuration, you can save it by clicking on the button Save. Saving is done simultaneously for the settings on all tabs. If you made an error you will get a popup with a warning and the data will not be saved. Locate the error, correct it and click OK again.

Click the button Cancel to ignore your changes and to close this window.

Defining orbs

The tabs for transits, secundary directions and symbolic directions contain an entry field for the orb, so you can define different orbs for these progressive techniques.

Points

Each tab shows the celestial points you can use for your progressive techniques. The points for primary directions will be explained later.

For each point you will find a checkbox. Check this box if you want to take the celestial point into account, deselect it if you do not want to use it.

See the paragraph Celestial Points Overview in the User Manual for an explanation for each point.

Time key

For symbolic directions, you can define a time key.

You can select one of the following options:

  • One degree: progress the points in the chart with exact one degree in longitude for each year of life.
  • Mean Sun: use the mean daily movement of the Sun and progress the points in the chart with this value, for the same amount of years.
  • True Sun: the real movement of the Sun in the first days of life define the progressions for the points in the chart in the same amount of years. Actually, this time key consists of the secondary Sun.

Primary directions

Primary directions are handled in a different way than the previous mentioned techniques.

You need to define a method, either Placidus or Regiomontanus, and also a time key. The appendix of the user manual contains descriptions for each time key.

You also have to select either Mundane (real positions) or Zodiacal (positions if celestial latitude is ignored).

There are two list of selectable celestial points:

  • Significators, the points that remain fixed as in the radix.
  • Promissors, the points that move because of the primary direction.

Currently, Enigma does not support aspects other than conjunctions and oppositions for primary directions. Also converse directions are not yet supported.

There is no orb to define as the results will be presented with the dates when the directions are exact.
