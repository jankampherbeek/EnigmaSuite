Search for chart

You can search for a chart by providing a part of the name. Enigma automatically uses wild cards, so do not add them. A search for John will recognize charts for John Lennon but also for Lyndon Johnson and Mary Bluejohn.

The search is case insensitive; Enigma makes no difference between lower case and upper case.

Leave the search term empty to find all charts.

The number of charts you can find is limited to 100.

After clicking the Search button, the results show in the table Search results. Click the chart you want to examine: the search window will close and the chart shows in the list of Available Charts in the main charts screen.
