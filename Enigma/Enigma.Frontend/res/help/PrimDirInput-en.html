Define a period for primary directions.

To calculate primary directions, you need to define a period. Enigma will calculate all primary directions that are formed in this period.

Please not that all dates are considered to be in the Gregorian calendar.

Conditions

The dates need to follow the following rules:

  • Use the format yyyy/mm/dd
  • The start date should be at least one day after the date of birth.
  • The end date should be later than the start date.

If you click Calculate Enigma will show the primary directions. If you made an error in the input, a popup-window is shown instead.
