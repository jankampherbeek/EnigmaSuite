Longitude equivalents

A longitude equivalent (LE) of a point is the longitude that corresponds with the declination if the point would have no latitude. For points that never have longitude, like the Sun and the lunar node, the LE is always the same as the real longitude. A special situation occurs when a planet is Out of Bounds (OOB): there are no longitudes that correspond to an OOB position. Usage is to use in such a case a fictive declination, called the co-declination. Please select the user manual for more information.
