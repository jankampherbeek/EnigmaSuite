Enigma Suite 1.0

Start screen

After starting Enigma Suite, you have four options:

  • Charts: calculating, analyzing and comparing charts.
  • Cycles: calculate cycles and other periodic functions.
  • Counts: statistics. Counting of occurrences.
  • Calculations: several astronomical calculations.

You can select each function by clicking one of the buttons with the same name or by clicking one of the images.

You can use only one of these options at the same time: this screen becomes inaccessible until you finish the option you are working on.

The Exit button closes Enigma Suite.
