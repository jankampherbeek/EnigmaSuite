Enigma - Progressive techniques

This is the start with screen for the calculation of progressive techniques. Currently, Enigma supports transits, secundary directions and symbolic directions.

Working with events

You need to define an event before you can use any of the progressive techniques. Events are saved in the database and the same event can be used for all techniques. An event is always coupled to the chart; you cannot define one event for several charts.

Under the header Available events you see all events that are currently available for the chart you are working on. Above the same header, Enigma shows the name for the chart and the currently selected event.

You need to select an available event - or create a new event and select it - before most of the functionality will become available.

Creating an event

You can define an event by clicking the button New event or by selecting the menu item Events - New Event.

Deleting an event

After selecting an event, you can delete it by clicking the button Delete event or by selecting the menu item Events - Delete event.

Configuration

For progressive techniques you use two configurations. The general configuration that is also used for working with charts in general, and an additional configuration for progressions.

Both configurations are accessible via the menu: General. Select either Configuration or Config progressions.

Progressive techniques

After selecting an event, you can calculate the positions for this event with one of the available techniques. Select one of the buttons Secundary dir., Transits or Symbolic dir. Alternatively, you can select one of the techniques in the menu underneath Techniques.
