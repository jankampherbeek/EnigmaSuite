Over de module Charts

De Charts module van Enigma ondersteunt de berekening, grafische weergave en analyse van horoscopen.

Je kunt horoscopen opslaan in een database om ze later weer te gebruiken.

Gebruik de help buttons, of de gebruikers handleiding, om meer te weten te komen over de mogelijkheden van Enigma.

Auteursrecht

Enigma is geschreven door by Jan Kampherbeek. Het programma is gratis en open source. Als je de code wilt gebruiken gelden wel enkele randvoorwaarden. Zie het bestand copyright.txt in de hoofd folder voor bijzonderheden.
