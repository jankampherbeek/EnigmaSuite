Aspects

This window contains one table that shows all aspects that are within orb. Enigma shows only the aspects that you defined in the configuration and also only the points (planets, angles etc.) in the same configuration.

The table comprises 8 columns:

  • Name of the first point.
  • Glyph of the first point (if available).
  • Name and distance of the actual aspect.
  • Glyph for the aspect.
  • Name of the second point.
  • Glyph of the second point (if available).
  • The actual orb.
  • The exactness of the aspect as a percentage. The higher the percentage, the smaller the orb.
