Input for an event

In this screen you define an event that can be analysed with progressive techniques.

Description. A descriptive text for the progressive event.

Location

The information about the location is optional. In most cases it would not make a difference. If you use parallax, the position of the Moon will be effected.

Name of location. A city name, address or other identification for the location. The default value is No location.

Longitude. Geographic longitude in the format ddd:mm:ss. For 123°45'30" you need to enter 123:45:30. You can skip the seconds, they will default to 0 seconds. So 145:15 is a valid indication for 145°15'00".

Use the scroll-down box to select either East or West.

Latitude. Geographic latitude in the format dd:mm:ss. For 52°13'30" you need to enter 52:13:30. You can skip the seconds, they will default to 0 seconds. So 45:15 is a valid indication for 45°15'00".

Use the scroll-down box to select either North or South.

Date and time

Date. Enter the date in the format yyyy:mm:dd, so June 16, 2022 would be entered as 2022/06/16.

Enigma assumes all dates for events are relatively recent. Therefore the Gregorian Calendar is used. If you really want to calculate the progressions for ancient periods, make sure to convert the Gregorian calendar to Julian and to apply the astronomical yearcount before the year 0.

Time. Enter the time in the format hh:mm:ss, using 24 hour notation. 2h38m30s PM would be 14:38:30. You can also leave the default time of 12:00.

TimeZone. Select one of the available timezones. The list contains the offset from UT (Greenwich Time) and the name of the timezone. If the time does not match with one of the available timezones, select LMT: Local Mean Time. You will get the possibility to define another offset from UT for the time used.

Apply DST. Check the box Apply DST if Daylight Saving Time is applicable.

Longitude for LMT. This field becomes available if you selected LMT as timezone. Here you can define the geographic longitude for the offset in degrees, minutes, and seconds, using the format ddd:mm:ss in the same way as for entering the geographic longitude for the location. Also select either East or West to indicate the direction for the offset.

Finish the definition of the event

Click the button Save to finish the definition of the event. If you made any errors, Enigma will show a popup with information about those errors. After you corrected the errors, you can again click Save.

Click the button Cancel if you do not want to continue.
