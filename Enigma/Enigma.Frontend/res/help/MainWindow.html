Enigma Astrology Research - Start screen

Projects

The screen shows a list of available projects. You can double-click on one of the projects to open it.

Alternatively you can open a project by double clicking the button Open project or by selecting the menu option Projects - Open project. This can come in handy if the number of projects is large.

To create a new project, click the button New project, or select the menu option Projects - New Project.

Data files

Projects always require a datafile. You can import a datafile by clicking the button Import data or selecting the menu option Data - Import data.

Datafiles can be used by several projects.

Periodicals

You do not need datafiles for periodicals. Currently you have the following possibilities:

  • Cycles: select the menu option Periodical - Cycles for a range of cycle calculations.
  • Ephemeris: select the menu option Periodical - Ephyemeris to construct a graphical or tabular ephemeris.
  • Occurrences: select the menu option Periodical - Occurrences to search for specific constellations.

Ad hoc actions

Use this option if you are not working on a project but just want to perform some calculations. Select one of the following:

  • Calculate chart: calculate a chart, show the graphic chart wheel, perform analyses and calculate progressions. This is a stripped down version of a standard astrological program.
  • Calculators: several calculations, coördinate conversions, time/calendar calculations etc.

Configuration and settings

Enigma supports two types of settings/configurations:

  • Settings for the applications, use the menu option General - Settings. Here you can define the location for your projects etc.
  • Astrological configuration, use the menu option General - Configuration. This is where you define the default astrological configuration, like orbs, supported celestial points, house system etc. In a project or ad hoc chart calculation, you an always overrule these configurations.
