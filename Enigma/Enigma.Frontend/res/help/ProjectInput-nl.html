Nieuw project

Voordat je een project kunt maken moet je eerst een data bestand importeren. Als dat nog niet is gebeurd kun je dat alsnog doen. Menu: Data - Importeer Data.

Beschrijving van het project

Om een project te definiëren moet je een naam en een beschrijving opgeven. Beide velden zijn verplicht.

  • In Naam project geef je de naam op die je wilt gebruiken voor je project.
  • In Beschrijving leg je aanvullende informatie over het project vast: een korte beschrijving van wat je wilt onderzoeken.

Controle groep

Als je een project definiëert maak je ook data aan voor een controle groep. Momenteel kun je alleen Standaard schuiven van plaats, datum en tijd selecteren. In toekomstige edities zal Enigma meerdere methoden ondersteunen voor het maken van controle groepen.

Vermenigvuldiging voor de controle groep

Kies één van de waarden voor de vermenigvuldigings factor voor de controle groep. Je kunt kiezen uit 1 (geen vermenigvuldiging), 10, 100 of 1000. Met een vermenigvuldigings factor vergroot je de controle groep.

Data bestand

Kies een data bestand uit de lijst met beschikbare data bestanden. Als de lijst leeg is moet je eerst een data bestand definiëren. Menu: Data - Importeer Data.

Het project opslaan

Als je op de knop Opslaan klikt, controleert Enigma je invoer en geeft eventuele fouten aan met een rode onderstreping. Als er geen fouten zijn, worden een folder structuur, een data bestand en een controle bestand gegenereerd. Als tijdens dit proces een fout ontstaat, krijg je een popup met een waarschuwing over deze fout. Als er geen fouten zijn krijg je een popup ter bevestiging. Als je deze popup sluit keer je terug naar het hoofd scherm.
